## Terminal

_This adds new theme to Terminal App of MacOS_

1. Download the [file](/assets/rick.terminal)
1. Import it to Terminal App

## Oh My Zsh

_This adds new theme to [Oh My Zsh](https://ohmyz.sh)_

1. Download the [file](/assets/rick.zsh-theme)
1. Place it under `$HOME/.oh-my-zsh/custom/themes` folder
1. Change `ZSH_THEME` to "rick" in `$HOME/.zshrc`

## Node.js

_Node.js version manager, [fnm](https://github.com/Schniz/fnm) is recommended_

1. Install using [Homebrew](https://brew.sh/)

   ```sh
   brew install fnm
   ```

1. To automatically run fnm use when a directory contains a .node-version or .nvmrc file, add the following to `$HOME/.zshrc` profile:

   ```sh
   eval "$(fnm env --use-on-cd)"
   ```

1. Usage

   [Available commands for an extended usage documentation](https://github.com/Schniz/fnm/blob/master/docs/commands.md)
